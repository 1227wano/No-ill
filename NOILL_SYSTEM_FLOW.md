# NOILL_SYSTEM_FLOW (독거노인 케어 솔루션: 노일이)

본 문서는 독거노인의 건강 모니터링, 정서적 지원 및 긴급 상황 대응을 위한 통합 솔루션 **No-ill(노일이)** 프로젝트의 시스템 아키텍처, 기능적 흐름 및 상세 명세를 정의합니다.

## 1. 프로젝트 개요
- **프로젝트명:** No-ill (노일이)
- **목적:** 독거노인의 거주지 내 생활 패턴 분석, 정서적 대화 서비스 및 응급 상황(낙상 등) 실시간 대응.
- **주요 대상:** 보호자가 직접 돌보기 어려운 환경의 독거노인 및 관련 지자체/기관.

## 2. 시스템 구성 (Architecture)

### 2.1 하드웨어 구성
| 구분 | 장치 | 역할 | 주요 사양 |
| :--- | :--- | :--- | :--- |
| **이동형 로봇** | Jetson Orin Nano | 사용자 추적, 순찰, 낙상 감지, 음성 대화 | LiDAR (X4-PRO), WebCam, 소형 디스플레이 |
| **고정 디스플레이** | Raspberry Pi 5 | 일정 관리, 복약 알림, 수면 분석, 영상 통화 | 터치 모니터, WebCam (BRIO100) |
| **중앙 서버** | AWS (예정) | LLM 처리, 로그 데이터 저장, 보호자 웹 관리 | REST API, WebRTC Signal Server |

### 2.2 네트워크 흐름
- **Device <-> Server:** WiFi를 통한 REST API 통신 (실시간 이벤트 및 로그 전송).
- **Display <-> Web:** WebRTC 기반 실시간 영상 및 음성 스트리밍.

## 3. 핵심 기능 명세 (Requirement Specification)

### 3.1 이동형 로봇: 노일이 (FR-N)
- **자율 주행 모드 전환:** - **추적 모드:** 객체 인식을 통해 사용자와 일정 거리를 유지하며 팔로잉.
  - **순찰 모드:** 사용자 미포착 시 구역 순찰 모드로 자동 전환.
  - **취침 모드:** 사용자 수면 확인 또는 명령 시 충전 스테이션 복귀.
- **이벤트 감지 및 대응:**
  - **낙상 및 고립 감지:** Pose Estimation을 통해 낙상 감지 시 즉시 서버에 사진 및 시간 정보 송신.
  - **음성 상호작용:** STT/TTS를 활용한 LLM 기반 대화 수행 및 일정/복약 정보 갱신.
- **정서 표현:** 소형 디스플레이를 통해 로봇의 감정 상태를 표정으로 표출.

### 3.2 고정형 거치 디스플레이 (FR-D)
- **정보 시각화:** 사용자의 일간 일정 및 복약 알림 대시보드 제공.
- **인터랙션:** 터치 기반 인터페이스 제공 및 '노일이' 호출 버튼 구현.
- **수면 패턴 분석:** 고정 위치에서 객체 인식을 통해 사용자의 수면 시간 측정 및 로봇과 상태 공유.
- **영상 통화:** 보호자와의 원터치 화상 통화 연결 (WebRTC).

### 3.3 보호자 웹 서비스 (FR-W)
- **로그 모니터링:** 수면 기록, 복약 이행 여부, 활동 로그 데이터를 카테고리별로 확인.
- **응급 알림:** 로봇이 전송한 위험 상황(사진 포함)을 실시간으로 확인 및 신고 대응.

## 4. 시스템 워크플로우 (Workflow)

### 4.1 일상 시나리오
1. **기상:** 디스플레이가 기상 시간 알림을 띄우고, 노일이가 충전기에서 나와 순찰 모드 시작.
2. **오전:** 노일이가 사용자를 찾아 추적 모드로 전환. 사용자와 대화하며 오늘 복약 일정 안내.
3. **오후:** 일정 정보를 확인하여 병원 방문 등 스케줄을 음성으로 재차 안내.

### 4.2 응급 상황 시나리오
1. **낙상 발생:** 노일이가 사용자의 낙상을 감지 (Pose Estimation).
2. **데이터 송신:** 즉시 현장 사진을 캡처하여 서버로 전송 및 보호자 웹에 경보 발생.
3. **사후 조치:** 보호자가 영상 통화를 연결하여 상태 확인 후 긴급 신고 절차 진행.

### 4.3 취침 시나리오
1. **수면 탐지:** 고정 디스플레이가 사용자의 수면 상태를 감지.
2. **모드 변경:** 서버를 통해 노일이에게 신호 전달 -> 노일이는 취침 모드(충전 복귀) 수행.
